    Ideology Classifier for Tweets


    🔍 Problem Context
    Understanding political sentiment and polarization on social media is key for researchers, policymakers, and analysts
    Classifying the ideological leanings of online messages helps detect trends, misinformation, and shifts in public opinion

    🛠 Solution Overview
    A supervised learning system for classifying tweets in Spanish into four political ideologies
    It processes tweet text and metadata using a combination of classical NLP techniques and machine learning models to identify ideological orientation with high accuracy

    🔧 Technical Description

    📄 Text Processing Preprocessing steps: tokenization, stop-word removal, lemmatization and filtering by PoS tags to retain only relevant terms Feature extraction via Bag of Words, TF-IDF, and embeddings (Word2Vec, FastText, BERT). 🧠 Classification Models evaluated include Naive Bayes, Logistic Regression, SVM, and LSTM networks These models are trained and evaluated across different feature representations to compare their classification performance 📊 Evaluation & Augmentation Data augmentation techniques are incorporated to increase dataset diversity and improve model generalization Performance evaluated using macro F1-score on labeled test data

    ✅ Use Cases
    • Political landscape monitoring by journalists or researchers
    • Sentiment trend analysis for social media platforms
    • Moderation support tools to detect ideological bias or polarization
    • Academic research in computational social science

    Smart Sewer Monitoring System with Ultra-Low Power Design


    🔍 Problem Context
    Urban flooding can cause severe damage to infrastructure and public safety. Traditional monitoring systems are expensive and reactive, lacking predictive insights A distributed, low-power solution is needed for early flood detection

    🛠 Solution Overview
    A wireless IoT platform that monitors water levels in urban sewers using low-energy sensors The data is sent to a central server, where an LSTM model predicts flood risk based on time series patterns.
    Users are alerted via mobile notifications in real time

    🔧 Technical Description

    📡 Data Collection Low-power ESP32 microcontrollers gather sensor data (water level, humidity, temperature) The collected data is formatted and transmitted in real time via the lightweight MQTT protocol to a central server for storage and processing 🧠 Predictive Modeling Sequential LSTM model trained on time-stamped CSV logs containing historical sensor data The model captures temporal dependencies that precede potential flood events, enabling early risk detection before critical thresholds are reached 📦 Visualization & Alerts A Real-time map-based interface (developed using Leaflet.js) shows sensor status and location When flood risk is detected by the model, the system sends instant alerts via the Telegram API to user devices


    ✅ Use Cases
    • City-scale flood monitoring and early warning systems
    • Emergency resource allocation during storms
    • Smart city integration with public safety platforms

    AR-ID: Augmented Reality Identification System


    🔍 Problem Context
    In various professional or educational environments (e.g. museums, warehouses, classrooms, factories), users interact with physical items that need fast identification, tracking, and visualization of related digital content
    Traditional systems use QR codes or barcodes, but these are limited in flexibility and user experience

    🛠 Solution Overview
    The AR-ID system proposes a more natural, camera-based interface for recognizing printed visual templates and reading associated text on them using classical computer vision and machine learning techniques
    Once the template is detected and the text is recognized (e.g. the name of a part or object), the system selects and projects a corresponding 3D model onto the real-world image, enabling intuitive augmented reality visualization

    🔧 Technical Description

    📐 Template Detection Detection of a planar object (e.g. printed sheet with a specific design) using keypoint matching and homography estimation Calculation of 3D pose relative to the camera from the template geometry and intrinsic parameters Projection of virtual 3D elements using the camera matrix 🧠 Text Recognition (OCR) Localizes the text area printed on the template Uses classical ML classifiers (e.g. KNN, Random Forest) trained on character images to recognize alphanumeric characters Implements line detection (RANSAC) and text reading for robust string reconstruction 📦 Model Selection & AR Rendering Maps recognized text (e.g. "CUBE") to a set of preloaded 3D models Selects the closest match using Levenshtein distance to tolerate OCR errors Overlays the 3D object on the template in real-world space


    ✅ Use Cases
    • Industrial part identification: Workers can point a camera at a label and instantly visualize a 3D model or exploded view of a mechanical part
    • Educational AR sheets: Students scan printed templates in textbooks or handouts to see interactive 3D structures (e.g. molecules)
    • Museum guides: Visitors use a mobile device to scan exhibit cards and see 3D reconstructions of ancient artifacts or artworks
    • Maintenance systems: Technicians scan equipment templates to get visual overlays of internal components or instructions
